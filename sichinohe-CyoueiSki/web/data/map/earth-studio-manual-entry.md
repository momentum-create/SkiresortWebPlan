# Earth Studio カメラ手入力（URL が効かないとき）

`@40.69908, 141.10739, 292a, ...` **だけでは視点は動きません。**
属性パネルに **数値を1つずつ** 入れるか、**フル URL** を使います。

## 方法 A — 属性パネル（確実）

右の **Attributes** → 値をダブルクリックして Enter:

### Camera Position

| 属性 | 値 |
|------|-----|
| Latitude | `40.6990810811416` |
| Longitude | `141.1073872446019` |
| Altitude | `292.067` |

### Camera Rotation

| 属性 | 値 |
|------|-----|
| Heading | `268.021` |
| Tilt | `78.410` |
| Roll | `0` |

> Tilt: **0°=真下、90°=水平**。78°はほぼ横からの斜め俯瞰。

## 方法 B — フル URL をアドレスバーに貼る

検索欄や `@...` 断片ではなく、**ブラウザの URL 欄**に以下をそのまま貼って Enter:

```
https://earth.google.com/studio/@40.6990810811416,141.1073872446019,292.06717126978504a,20y,268.02066132841924h,78.41026408687615t
```

※ カンマの **直後にスペースを入れない**。

## 方法 C — Viewport にペースト

1. 中央の **地球ビュー** をクリック（青い枠 = フォーカス済み）
2. **Ctrl+V**（Mac は Cmd+V）で **フル URL** を貼る
3. 検索バーや緯度欄には貼らない

## うまくいかないとき

- Google Earth（デスクトップ）ではなく **earth.google.com/studio** か確認
- 既存プロジェクトでキーフレームが有効だと上書きされない → フレームを選んでから属性を変更
- まず検索で `40.69839, 141.099714` と七戸付近へ飛んでから、上の数値を入れる

反映後: `node parse-earth-studio-url.mjs --url="（Copy Location URL）"` で `camera.json` に同期。

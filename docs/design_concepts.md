# スキー場Webテンプレート — デザインコンセプト（3案 + v2）

> **resort-ux-designer** | v2: 2026-06-07  
> 目的: モバイルファースト・アワード級UI/UX・全国スキー場テンプレート化  
> **初回適用リゾート**: 七戸町営スキー場（パレット・コピー基準）

---

## 共通要件（3案すべて）

| 項目 | 方針 |
|------|------|
| ブレークポイント | モバイル 375px 基準 → 768px タブレット → 1280px デスクトップ |
| タッチターゲット | 最小 44×44px |
| コントラスト | WCAG AA（本文 4.5:1 以上） |
| データ分離 | テキスト・画像URL・料金は `data/` JSON/TS に集約 |
| コンバージョン導線 | ヒーロー → 本日の状況 → チケット → アクセス の4段 |

---

## 案A: 「Avalanche Immersion（雪崩的没入）」

### コンセプト
雪山の壮大さを**シネマティック全画面体験**で伝える。スクロールするたびに視差（パララックス）と粒子アニメーションが雪原の奥行きを演出し、来場者を「すでにゲレンデにいる」感覚へ引き込む。

### 構成（モバイル）

```
┌─────────────────────────┐
│ 全画面ヒーロー動画/画像    │  ← 100dvh、下部にグラデーション
│ リゾート名（大見出し）     │
│ 本日の積雪・天気ピル       │
│ [本日の運行] [チケット]    │  ← 固定CTAペア
├─────────────────────────┤
│ 横スワイプ ライブカム帯    │
├─────────────────────────┤
│ 縦スクロール コースカード  │  ← フル幅、角丸24px
├─────────────────────────┤
│ 料金比較（3プラン）        │
├─────────────────────────┤
│ アクセス（地図サムネ）     │
└─────────────────────────┘
│ 固定ボトムナビ（5アイコン）│
└─────────────────────────┘
```

### カラーパレット

| トークン | 値 | 用途 |
|----------|-----|------|
| `--snow` | `#F4F7FB` | 背景・雪面ハイライト |
| `--glacier` | `#0A2540` | ヘッダー・ヒーロー下地 |
| `--ice` | `#4FC3F7` | アクセント・リンク |
| `--powder` | `#E8EEF4` | カード背景 |
| `--ember` | `#FF6B35` | CTA・緊急告知 |
| `--night` | `#060D18` | フッター |

### タイポグラフィ
- 見出し: **Syne**（幾何学的・冒険的）
- 本文: **Noto Sans JP**（可読性・日本語対応）

### アニメーション指定（Framer Motion）

| 要素 | 動き | パラメータ |
|------|------|------------|
| ヒーロー | ゆっくりズームイン | `scale: 1→1.08`, 12s loop, ease linear |
| 雪粒子 | Canvas/CSS 粒子 | 60fps 目標、prefers-reduced-motion で停止 |
| セクション入場 | 下からフェード | `y: 24→0`, `opacity: 0→1`, 0.5s, stagger 0.08s |
| CTA | 微パルス | `scale: 1→1.02`, 2s loop（チケットボタンのみ） |
| ライブステータス | 数値カウントアップ | 0.8s ease-out |

### 強み
- ブランド印象・SNSシェア映えが最強
- 映像リッチなリゾート向き

### 弱み
- 低スペック端末でパフォーマンス負荷
- 情報密度が低く、料金・アクセスへの導線が弱い
- 小規模ゲレンデでは「盛りすぎ」に見えるリスク

## 案B: 「Alpine Clarity（アルパイン・クラリティ）」

### コンセプト
スイス・アルプス系リゾートサイトの**編集デザイン＋情報設計**をモバイルに最適化。余白とタイポグラフィで信頼感を作り、**コンバージョン（チケット購入・アクセス）を最短3タップ**に収束させる。

### 構成（モバイル）

```
┌─────────────────────────┐
│ スティッキーヘッダー       │  ← ロゴ + ハンバーガー
├─────────────────────────┤
│ ヒーロー（65dvh）         │  ← 写真＋オーバーレイテキスト
│ ステータスチップ行         │  ← 積雪/リフト/天気（横スクロール）
├─────────────────────────┤
│ プライマリCTAバンド        │  ← 2列: チケット | 本日の運行
├─────────────────────────┤
│ Bentoグリッド             │  ← 2×2: コース/レッスン/宿泊/イベント
├─────────────────────────┤
│ 料金カード（縦積み）       │  ← 比較しやすい表形式
├─────────────────────────┤
│ ニュース（3件）            │
├─────────────────────────┤
│ アクセス＋営業時間         │
└─────────────────────────┘
```

### カラーパレット

| トークン | 値 | 用途 |
|----------|-----|------|
| `--canvas` | `#FAFBFC` | ページ背景 |
| `--ink` | `#1A1F2E` | 見出し・本文 |
| `--slate` | `#5C6678` | 補助テキスト |
| `--alpine` | `#1B6B93` | プライマリアクセント |
| `--alpine-soft` | `#E3F2F8` | バッジ・チップ背景 |
| `--gold` | `#C9A227` | プレミアム・早割バッジ |
| `--success` | `#2D8F5E` | 運行中・オープン |
| `--danger` | `#C0392B` | クローズ・注意 |

### タイポグラフィ
- 見出し: **DM Sans**（モダン・国際的）
- 本文: **Noto Sans JP**
- 数字・料金: **Tabular nums**, `font-feature-settings: "tnum"`

### アニメーション指定

| 要素 | 動き | パラメータ |
|------|------|------------|
| ページ入場 | コンテナフェード | `opacity: 0→1`, 0.3s |
| Bentoカード | ホバー/タップリフト | `y: 0→-4px`, shadow 増加, 0.2s |
| ステータスチップ | スライドイン | `x: -16→0`, stagger 0.05s |
| アコーディオン（FAQ） | 高さアニメ | `height: auto`, spring stiffness 300 |
| スクロール進捗 | トップバー | `scaleX: 0→1`, scroll-linked |

### 強み
- モバイルUX・情報探索・コンバージョンのバランスが最良
- 全国規模の小〜大リゾートにそのまま流用可能
- パフォーマンス負荷が低く、LCP 2.5s 以内を達成しやすい

### 弱み
- 映像派リゾートでは「地味」に感じる可能性
- 差別化には写真・コピーの質に依存

## 案C: 「Powder Pulse（パウダー・パルス）」

### コンセプト
**ゲーミフィケーション＋ライブデータ**で「今この瞬間の雪山」を体験化。Snap風の縦スワイプセクションと太いグラデーションUIで、Z世代・SNS世代の来場者をターゲットにする。

### 構成（モバイル）

```
┌─────────────────────────┐
│ フルブリード グラデ背景    │
│ 巨大タイポ リゾート名      │
│ ライブカウンター（来場者数）│
├─────────────────────────┤
│ スナップカード縦スワイプ    │  ← コース別フルスクリーン
│ （左右スワイプで切替）     │
├─────────────────────────┤
│ フローティングCTA円形      │  ← 右下固定、チケット
├─────────────────────────┤
│ 料金（タブ切替）           │  ← 平日/休日/シーズン券
├─────────────────────────┤
│ SNSフィード風ギャラリー    │
└─────────────────────────┘
│ ボトムシートナビ           │  ← スワイプで展開
└─────────────────────────┘
```

### カラーパレット

| トークン | 値 | 用途 |
|----------|-----|------|
| `--void` | `#0D0221` | 深い紫黒背景 |
| `--neon-cyan` | `#00F5FF` | グラデ起点 |
| `--neon-magenta` | `#FF00AA` | グラデ終点 |
| `--frost` | `#FFFFFF` @ 12% opacity | ガラスカード |
| `--lime` | `#B8FF3C` | ハイライト・バッジ |
| `--surface-glass` | `rgba(255,255,255,0.08)` | ガラスモーフィズム |

### タイポグラフィ
- 見出し: **Clash Display** または **Space Grotesk**（大胆・未来的）
- 本文: **Noto Sans JP**

### アニメーション指定

| 要素 | 動き | パラメータ |
|------|------|------------|
| 背景 | グラデーションシフト | hue rotate 15°, 8s loop |
| スナップカード | スワイプスナップ | `scroll-snap-type: y mandatory` |
| ライブカウンター | 数字ロール | odometer風、0.6s |
| FAB | バウンス | `y: 0→-6→0`, 1.5s loop |
| ガラスカード | ぼかし＋境界光 | `backdrop-blur: 16px`, border gradient |

### 強み
- 若年層・インフルエンサー向けPRに強い
- 「今来るべき理由」が視覚的に伝わる

### 弱み
- ファミリー・シニア層には読みにくい（コントラスト・文字サイズ）
- ガラスモーフィズム＋グラデは汎用テンプレートとして飽きやすい
- アクセシビリティ達成コストが高い
- 地方の小規模ゲレンデのブランドトーンと不一致になりやすい

## v2 ベンチマークボード（参照 3 サイト）

各サイトから **5 要素を盗む**（コピーではなく編集デザインとして再解釈）。

### 参照 1: [LAAX](https://www.laax.com)（スイス・ゲレンデマップ／リゾート）

| # | 盗む要素 | 七戸テンプレへの適用 |
|---|----------|---------------------|
| 1 | ライブステータスが **横一列の大型メトリクス** | `LiveStatusStrip` をチップ列 → 3 カラムバーへ |
| 2 | 地図が体験の中心（P0） | トップ CTA・Bento 先頭を `/map` 誘導 |
| 3 | 落ち着いた **スノー×アルパイン** パレット | 七戸パレット（下記）の `--alpine-soft` 基調 |
| 4 | Display タイポの **ジャンプ率** | Syne + 本文 Noto |
| 5 | 余白を「贅沢に」使うセクションリズム | `py-16` / `py-24` 厳守 |

### 参照 2: [ニセコリゾート観光協会](https://www.niseko.ne.jp)（日本・町営〜中規模）

| # | 盗む要素 | 七戸テンプレへの適用 |
|---|----------|---------------------|
| 1 | **実写ヒーロー**＋短いキャッチ 2 行 | 権利済み SVG/実写、`overlayLines` 2 行 |
| 2 | 家族・初級者への **安心トーン** | コピー・バッジを「やさしい中斜面」寄り |
| 3 | チケット導線の **明確な 2 列 CTA** | `PrimaryCtaBand` 維持 |
| 4 | ニュース＋雪質レポートの **情報階層** | News + LiveStatus の分離 |
| 5 | 日本語本文の **行長・行間**（1.75） | `globals.css` body 維持 |

### 参照 3: Awwwards — Resort / Travel 部門（例: 雪山×編集デザイン系）

| # | 盗む要素 | 七戸テンプレへの適用 |
|---|----------|---------------------|
| 1 | **非対称 Bento**（1 枚だけ大きい） | `BentoExploreGrid` index 0 = 2×2 span |
| 2 | 数値の **tabular + mono** | IBM Plex Mono on 積雪・料金 |
| 3 | スクロール reveal の **stagger** | `useScrollReveal` 0.08s |
| 4 | ヒーロー **微ズーム**（動画なし） | scale 1→1.04, 8s |
| 5 | **絵文字アイコン禁止**、SVG のみ | BottomNav / フィルタ |

## 七戸町営スキー場パレット（v2 確定）

青森・七戸の **林（forest）× 雪（snow）× 八甲田・津軽の寒色** をトークン化。トップはライト基調、マップは `--alpine-dark` で接続。

| トークン | 値 | 意味 |
|----------|-----|------|
| `--canvas` | `#F7F9FB` | 雪面ハイライト・ページ背景 |
| `--ink` | `#1C2434` | 見出し・本文 |
| `--slate` | `#5A6578` | 補助テキスト |
| `--forest` | `#2D5A4A` | 七戸の森・町営の落ち着き |
| `--alpine` | `#2D6B7A` | 寒色アクセント（マップと共有） |
| `--alpine-soft` | `#E8F3F0` | ステータスバー・チップ背景 |
| `--alpine-dark` | `#1A4D42` | マップ UI・ヒーローグラデ下部 |
| `--gold` | `#B8862B` | 早割・プレミアム（秋もみじ金） |
| `--success` | `#2D8F5E` | 運行中 |
| `--danger` | `#C0392B` | クローズ |

**マップ連携**: `sichinohe-CyoueiSki/web` の `#0c1220` を将来的に `--alpine-dark` 系へ寄せる（`resort-map-bridge`）。
